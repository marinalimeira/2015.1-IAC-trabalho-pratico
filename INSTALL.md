#Instruções para compilação e execução do programa

## Compilação
```
gcc src/main.c -o bin
```

## Execução

### Utilização intensa de CPU
```
./bin "cpu"
```

### Utilização intensa de CPU e memória
```
./bin "cpu-mem"
```

### Geração de gráficos
* É necessário possuir python e matplotlib instalados.
```
python results.py 
```
